# UpTech Bot - Telegram Bot для Обучения

Telegram bot для платформы UpTech Academy с функциями просмотра курсов, материалов, событий и информации о компании.

## Функционал

- 📚 **Курсы** - Выбор из 3 курсов (Трансформеры, Python для ИНС, Sage и LaTeX)
- 📖 **Полезные материалы** - Ссылки на ресурсы для обучения
- 📅 **События** - Список предстоящих вебинаров
- ℹ️ **О нас** - Информация о компании
- 📥 **Скачивание материалов** - PDF, видео, домашние задания, примеры кода

## Установка

### 1. Клонировать репозиторий или распаковать файлы

```bash
cd uptech_bot
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
```

### 3. Активировать виртуальное окружение

**Windows:**
```bash
.venv\Scripts\activate
```

**Linux/Mac:**
```bash
source .venv/bin/activate
```

### 4. Установить зависимости

```bash
pip install -r requirements.txt
```

### 5. Получить Bot Token

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен
4. Скопируйте токен и вставьте его в `config.py`

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

### 6. Запустить бота

```bash
python main.py
```

## Использование

1. Откройте Telegram и найдите вашего бота
2. Отправьте `/start` для запуска
3. Выберите интересующие вас опции из меню:
   - 📚 **Курсы** - выберите курс и скачивайте материалы
   - 📖 **Полезные материалы** - список полезных ресурсов
   - 📅 **События** - список предстоящих вебинаров
   - ℹ️ **О нас** - информация о компании

## Добавление новых материалов

📖 **[Полная инструкция в файле ДОБАВЛЕНИЕ_МАТЕРИАЛОВ.md](ДОБАВЛЕНИЕ_МАТЕРИАЛОВ.md)**

### Быстрый пример

Отредактируйте файл `data/courses.json` и добавьте материалы к курсу:

```json
"materials": [
  {
    "type": "pdf",
    "name": "Лекция 1: Основы",
    "url": "file:///C:/Users/admin/OneDrive/Desktop/uptech_bot/materials/lecture_1.pdf"
  },
  {
    "type": "video",
    "name": "Видеолекция",
    "url": "https://youtube.com/watch?v=example"
  },
  {
    "type": "homework",
    "name": "Домашнее задание 1",
    "url": "file:///C:/Users/admin/OneDrive/Desktop/uptech_bot/materials/homework_1.pdf"
  }
]
```

### Типы материалов

- `pdf` - PDF документы и конспекты
- `video` - Видео уроки (YouTube или локальные ссылки)
- `homework` - Домашние задания
- `code` - Примеры кода
- `other` - Другие материалы
