# 🚀 БЫСТРЫЙ СТАРТ: Деплой на Railway в 5 минут

## 📋 Что вам понадобится:
- ✅ GitHub аккаунт (создайте на https://github.com)
- ✅ Railway аккаунт (создайте на https://railway.app)
- ✅ Git установлен на компьютер (https://git-scm.com)
- ✅ BOT_TOKEN от @BotFather

---

## 🔧 Шаг 1: Инициализация Git репозитория

Откройте PowerShell и выполните:

```powershell
cd C:\Users\admin\OneDrive\Desktop\uptech_bot

# Инициализируем Git
git init

# Добавляем все файлы
git add .

# Создаем первый коммит
git commit -m "Initial commit: UpTech Bot"

# Переименовываем ветку на main
git branch -M main
```

---

## 📤 Шаг 2: Загрузка на GitHub

1. Откройте https://github.com/new
2. Заполните:
   - **Repository name**: `uptech-bot`
   - **Description**: `Telegram Bot for UpTech Academy`
   - **Public** ✅ (важно!)
3. Нажмите `Create repository`

4. В PowerShell выполните (замените YOUR_USERNAME и YOUR_TOKEN):

```powershell
git remote add origin https://github.com/YOUR_USERNAME/uptech-bot.git
git push -u origin main
```

При запросе токена:
- Username: ваше имя пользователя GitHub
- Password: [Personal Access Token](https://github.com/settings/tokens)

---

## 🚂 Шаг 3: Развертывание на Railway

1. Откройте https://railway.app
2. Нажмите `Login` → авторизуйтесь через GitHub
3. Нажмите `+ New Project`
4. Выберите `Deploy from GitHub repo`
5. Выберите репозиторий `uptech-bot`
6. Railway автоматически:
   - Обнаружит `Procfile`
   - Установит зависимости из `requirements.txt`
   - Запустит `python main.py`

---

## 🔐 Шаг 4: Добавленние BOT_TOKEN

1. На странице Railway проекта нажмите на сервис
2. Перейдите на вкладку `Variables`
3. Нажмите `+ New Variable`
4. Добавьте:
   ```
   BOT_TOKEN = YOUR_TOKEN_FROM_BOTFATHER
   ```
5. Нажмите `Deploy`

---

## ⚠️ Важные замечания

1. **Никогда не коммитьте токен на GitHub!** 
   - Токен хранится в переменной окружения `BOT_TOKEN`
   - Файл `.env` в `.gitignore`

2. **Railway дает $5 кредитов в месяц** (этого достаточно)

3. **Если Bot упадет**, Railway автоматически перезагрузит его

4. **Могут быть задержки в обновлении** - максимум несколько минут

---

## 🆘 Если что-то не работает

1. **Проверьте логи** на Railway в вкладке `Logs`
2. **Проверьте переменную** `BOT_TOKEN` заполнена правильно
3. **Убедитесь**, что GitHub репозиторий **Public**
4. **Перезапустите** сервис нажав на кнопку `Redeploy`
