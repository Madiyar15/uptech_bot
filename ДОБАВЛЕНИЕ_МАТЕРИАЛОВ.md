# Инструкция по добавлению материалов к курсам

## Структура файла courses.json

Файл `data/courses.json` содержит информацию о курсах и их материалах.

Каждый курс имеет структуру:

```json
{
  "id": 1,
  "name": "Название курса на русском",
  "name_en": "Course name in English",
  "lecturer": "Имя лектора",
  "description": "Полное описание курса...",
  "level": "Начинающий",
  "duration": "8 недель",
  "price": "Бесплатно",
  "materials": []
}
```

## Добавление материалов

### Структура материала

```json
{
  "type": "тип_материала",
  "name": "название файла",
  "url": "путь_или_url_к_файлу"
}
```

### Типы материалов

- `pdf` - PDF документы и конспекты
- `video` - Видео уроки (YouTube или локальные ссылки)
- `homework` - Домашние задания
- `code` - Примеры кода
- `other` - Другие материалы

### Примеры добавления

#### 1. Локальный PDF файл

```json
{
  "type": "pdf",
  "name": "Лекция 1: Основы трансформеров",
  "url": "file:///C:/Users/admin/OneDrive/Desktop/uptech_bot/materials/lecture_1.pdf"
}
```

📌 **Важно:** Используйте формат пути с прямыми слешами `/` и `file:///`

#### 2. YouTube видео

```json
{
  "type": "video",
  "name": "Видеолекция: Введение в ИНС",
  "url": "https://youtube.com/watch?v=dQw4w9WgXcQ"
}
```

#### 3. Файл примера кода

```json
{
  "type": "code",
  "name": "Пример: Создание трансформера",
  "url": "file:///C:/Users/admin/OneDrive/Desktop/uptech_bot/code_examples/transformer.py"
}
```

## Быстрое редактирование JSON

### Способ 1: VS Code

1. Откройте файл `data/courses.json` в VS Code
2. Найдите нужный курс (по `id`)
3. Добавьте объекты материалов в массив `materials`
4. Сохраните файл (Ctrl+S)

### Способ 2: Онлайн JSON редактор

1. Скопируйте содержимое файла
2. Вставьте на https://jsoncrack.com/ или https://www.jsonlint.com/
3. Отредактируйте и проверьте синтаксис
4. Скопируйте обратно в файл

### Способ 3: Python скрипт

Создайте файл `add_materials.py`:

```python
import json

def add_material_to_course(course_id, material):
    """Добавляет материал к курсу"""
    with open('data/courses.json', 'r', encoding='utf-8') as f:
        courses = json.load(f)
    
    for course in courses:
        if course['id'] == course_id:
            course['materials'].append(material)
            break
    
    with open('data/courses.json', 'w', encoding='utf-8') as f:
        json.dump(courses, f, ensure_ascii=False, indent=2)
    
    print(f"Материал добавлен к курсу {course_id}")

# Пример использования
material = {
    "type": "pdf",
    "name": "Новая лекция",
    "url": "file:///path/to/file.pdf"
}

add_material_to_course(1, material)
```

Запустите: `python add_materials.py`

## Проверка синтаксиса JSON

После редактирования проверьте синтаксис:

```python
import json

try:
    with open('data/courses.json', 'r', encoding='utf-8') as f:
        json.load(f)
    print("✅ JSON файл в порядке!")
except json.JSONDecodeError as e:
    print(f"❌ Ошибка в JSON: {e}")
```

## Советы

- ✅ Используйте **полные пути** к файлам (не относительные)
- ✅ На Windows используйте формат: `file:///C:/path/to/file`
- ✅ На Linux/Mac используйте формат: `file:///home/user/path/to/file`
- ✅ Всегда **проверяйте JSON синтаксис** после редактирования
- ✅ **Сохраняйте файлы** перед перезапуском бота
- ❌ Не забывайте **запятые** между элементами массива
- ❌ Не используйте **обратные слеши** `\` в путях (только прямые слеши `/`)
